## "Числа по порядку"

### Задание

Дано три целые числа. 

Упорядочите их в порядке возрастания.

![up](img/up.png)

### Формат входных данных

Даны три целые числа.

### Формат выходных данных

Вывести все числа от меньшего к большему.

#### Примечание

Предоставленную заготовку не менять. Ваша задача, написать алгоритм, который распределит значения трех переменных так, что в "a" будет наименьшее значение, а в "c" наибольшее.

### Решение задачи

```python
a = int(input("a: "))
b = int(input("b: "))
c = int(input("c: "))

# TODO: you code here...

print(a, b, c)
```

вариант1:

a = int(input("a: "))
b = int(input("b: "))
c = int(input("c: "))
if a >= b:
    if b >= c:
        print(c, b, a)
    elif c >= a:
        print(b, a, c)
    else:
        print(b, c, a)
else:
    if c >= b:
        print(a, b, c)
    elif c >= a:
        print(a, c, b)
    else:
        print(c, a, b)

вариант2:

a = int(input("a: "))
b = int(input("b: "))
c = int(input("c: "))
if b <= a:
    a, b = b, a
if c <= a:
    a, c = c, a
if c <= b:
    c, b = b, c
print(a, b, c)
